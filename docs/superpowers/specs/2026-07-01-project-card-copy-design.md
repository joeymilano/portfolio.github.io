# 首页项目卡片文案调整

## 目标

修正 WeNeed 与叙事卡牌 RPG 的首页定位，让文案准确描述产品机制，并突出 Joey 作为产品设计师运用 Agentic Engineering 将复杂系统落地的能力。

## 文案

### WeNeed

- 中文：短视频闲置物品赠送 + We 积分
- 英文：Short-video item gifting + We Points.

### 叙事卡牌 RPG

- 中文：Agentic Engineering 思维开发的卡牌肉鸽游戏
- 英文：A card roguelike built with an Agentic Engineering mindset.

## 修改范围

仅修改 `index.html` 中两张项目卡片的默认英文文本，以及中英文语言切换字典中的对应文本。不调整标题、标签、指标、布局或项目详情页。

## 验收标准

- 首次加载英文页面时显示新英文文案。
- 切换中文后显示用户确认的新中文文案。
- 两张卡片的其他内容和交互保持不变。

# Joey 内容分发 Playbook

## 母内容原则

完整文章先发布于 `joeyzhao.cc/writing/`。其他平台使用原生改编，不复制整篇正文。每个平台版本保留一个明确观点和一个回到母文或相关作品案例的入口。

## 账号与角色

- LinkedIn：专业摘要、项目复盘、精选英文版。
- X：短观点、thread、Build in Public 实时记录。账号：`https://x.com/JoeyKonad`。
- 小红书：中文图文与日常设计观察。账号：`https://www.xiaohongshu.com/user/profile/6279c0b50000000021021021`。
- 视频号：60–90 秒口播/拆解。名称：`Joey的产品美学`，ID：`sphmQz03lemSzBD`。

## 回流链接

统一使用：

```text
https://joeyzhao.cc/writing/{slug}.html?utm_source={linkedin|x|xiaohongshu|wechat_channels}&utm_medium=social&utm_campaign={slug}
```

LinkedIn 和 X 可直接放链接。小红书、视频号若正文外链受限，则用主页可访问入口、置顶内容或口播引导，不使用规避平台规则的表达。

## 一篇文章的派生包

1. 网站：完整论证、来源、边界、相关作品链接。
2. LinkedIn：一个职业判断 + 三个证据点 + 回链。
3. X：5–8 条 thread；第一条只保留最强观点，最后一条回链。
4. 小红书：6 张图；现象、反常识、机制、边界、总结、互动。
5. 视频号：3 秒钩子、45 秒拆解、15 秒个人判断、结尾引导查看主页。

## 发布后记录

按文章 campaign 记录：网站访问、作品案例点击、联系按钮点击、各平台收藏/转发/评论。优先复用能同时带来网站访问和作品浏览的主题，而不是只看点赞。
